

Tech Stack
------------

- Java 1.8+
- Maven 3.5+
- Lombok
- Hibernate Validator
- JUNIT 5

Testing Application
---------------
1. Open application through Intelij Idea
2. Let download necessary dependencies to be run the application
3. Right click on the Project and select 'Run All Tests' option.
4. All tests will be executed.
